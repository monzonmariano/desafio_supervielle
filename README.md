# Desafío Técnico - API REST (Banco Supervielle)

API RESTful desarrollada en Java como parte de un desafío técnico.

El proyecto implementa la lógica de negocio para un CRUD de usuarios /personas  siguiendo el patrón de diseño MVC. Utiliza Spring Framework para la gestión de los controladores y servicios, y Spring Data JPA / Hibernate para la capa de persistencia.

La aplicación fue configurada para conectarse a una base de datos MySQL y desplegada en la plataforma en la nube Heroku.

---

## 🚀 Stack de Tecnologías

* **Lenguaje:** Java (8+)
* **Framework:** Spring Framework (MVC, Boot)
* **Persistencia (ORM):** Spring Data JPA, Hibernate
* **Base de Datos:** MySQL
* **Deploy:** Heroku
